🌱 Explainable AI Based Soil Fertility Prediction

📌 Project Overview

This project predicts soil fertility using Machine Learning techniques and provides clear explanations using Explainable AI (XAI). It helps users understand how different soil parameters influence fertility.

 🚀 Technologies Used

* Python
* Machine Learning
* Random Forest Algorithm
* Explainable AI (XAI)
* Pandas, NumPy, Scikit-learn

## 📊 Features

* Predicts soil fertility based on input values
* Uses trained Random Forest model
* Provides explanation for predictions (XAI)
* Simple and efficient implementation

## 📁 Project Structure

* `app.py` → Main application file
* `fertilizer.ipynb` → Model training and analysis
* `dataset1.csv` → Dataset used for training
* `optimized_rf_model.pkl` → Optimized trained model
* `soil_fertility_rf_model.pkl` → Final trained model

## ⚙️ How to Run the Project

1. Install required libraries:
   pip install -r requirements.txt
2. Run the application:
   python app.py
